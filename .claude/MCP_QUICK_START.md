# MCP 快速开始指南

## 🚀 立即使用（3 步）

### 1️⃣ 重启 Claude Code
**VSCode**: `Cmd+Shift+P` → "Reload Window"

### 2️⃣ GitHub 认证
```bash
/mcp
# 选择 "Authenticate" for GitHub
# 在浏览器中完成 GitHub 登录
```

### 3️⃣ 开始使用
```
"数据库中有多少支股票？"
"列出所有 open 状态的 Issues"
```

## ⚠️ 注意事项

1. **首次使用需认证**: GitHub MCP 需要 OAuth 认证
2. **数据库权限**: 当前配置有完整读写权限
3. **网络依赖**: GitHub MCP 需要网络连接
4. **速率限制**: GitHub API 有调用频率限制
