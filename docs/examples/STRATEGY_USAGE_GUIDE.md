# 趋势+动能双重确认策略 - 使用指南

## 策略简介

这是一个基于 **EMA 趋势过滤 + MACD 动能触发** 的双重确认入场策略。

### 核心理念
**顺势而为**：只有在大趋势向上时,才在动能转强的瞬间入场。

### 策略优势
- ✅ 双重确认,降低假信号
- ✅ 趋势+动能共振,提高胜率
- ✅ 清晰的入场和离场条件
- ✅ 参数可调,适应不同市场环境

---

## 指标组合

### 趋势指标: EMA (指数移动平均线)
- **EMA 20**: 短期趋势
- **EMA 60**: 中期趋势
- **EMA 120**: 长期趋势

### 动能指标: MACD
- **快线周期**: 12
- **慢线周期**: 26
- **信号线周期**: 9

---

## 入场逻辑

**买入条件 (必须同时满足)**:

1. **趋势过滤** (长期趋势向上)
   - 股价 > EMA 120

2. **多头排列** (短中期趋势强化)
   - EMA 20 > EMA 60

3. **动能触发** (爆发力捕捉)
   - MACD 在零轴上方发生"金叉" **OR**
   - MACD 柱状图由负转正

### 入场逻辑示意图
```
价格位置:     close > EMA120 ✓
趋势排列:     EMA20 > EMA60 ✓
动能信号:     MACD金叉(零轴上方) ✓
                    ↓
              【买入信号】
```

---

## 离场逻辑

**卖出条件 (满足任一即离场)**:

1. **快速离场**: MACD 死叉
2. **趋势反转**: 股价跌破 EMA 60

## 参数配置

策略提供以下可调参数:

### 通用参数
| 参数名 | 默认值 | 说明 |
|--------|--------|------|
| `top_n` | 20 | 每期选择前N只股票 |
| `holding_period` | 5 | 最短持仓期(天) |
| `rebalance_freq` | 'W' | 调仓频率(W=周) |

### EMA 趋势参数
| 参数名 | 默认值 | 范围 | 说明 |
|--------|--------|------|------|
| `ema_short` | 20 | 10-50 | 短期EMA周期 |
| `ema_mid` | 60 | 30-120 | 中期EMA周期 |
| `ema_long` | 120 | 60-250 | 长期EMA周期 |

### MACD 动能参数
| 参数名 | 默认值 | 范围 | 说明 |
|--------|--------|------|------|
| `macd_fast` | 12 | 5-20 | MACD快线周期 |
| `macd_slow` | 26 | 15-40 | MACD慢线周期 |
| `macd_signal` | 9 | 5-15 | MACD信号线周期 |

### 入场条件参数
| 参数名 | 默认值 | 说明 |
|--------|--------|------|
| `require_macd_above_zero` | True | 要求MACD在零轴上方 |
| `enable_histogram_trigger` | True | 启用柱状图触发 |
| `require_ema_alignment` | True | 要求EMA多头排列 |

### 离场条件参数
| 参数名 | 默认值 | 说明 |
|--------|--------|------|
| `exit_on_macd_cross` | True | MACD死叉离场 |
| `exit_on_ema_break` | True | 跌破EMA60离场 |

### 风险控制参数
| 参数名 | 默认值 | 说明 |
|--------|--------|------|
| `min_volume_ratio` | 0.8 | 最小成交量倍数 |

## 策略评分逻辑

策略使用综合评分对股票进行排序:

### 评分维度

1. **趋势强度** (权重 60%)
   - EMA20 > EMA60: +0.3分
   - EMA60 > EMA120: +0.3分
   - 价格 > EMA120: +0.4分

2. **动能强度** (权重 40%)
   - MACD > 0: 分数 = MACD值/10 (归一化到0-1)

### 综合评分公式
```
总分 = 趋势强度 × 0.6 + 动能强度 × 0.4
```

最终选择评分最高的 `top_n` 只股票买入。

## 使用建议

### 适用市场环境
- ✅ **牛市**: 效果最佳,趋势明确
- ✅ **震荡市**: 中等效果,需调高入场阈值
- ❌ **熊市**: 不推荐,大趋势向下

### 参数调优建议

#### 保守配置 (降低风险)
```python
{
    'ema_long': 200,              # 更长的趋势过滤
    'require_macd_above_zero': True,  # 严格要求零轴上方
    'require_ema_alignment': True,    # 严格要求多头排列
    'top_n': 15,                  # 精选少量股票
}
```

#### 激进配置 (增加机会)
```python
{
    'ema_long': 60,               # 更短的趋势过滤
    'require_macd_above_zero': False, # 允许零轴下方金叉
    'enable_histogram_trigger': True, # 启用柱状图触发
    'top_n': 30,                  # 增加持仓数量
}
```

#### 短线配置 (快进快出)
```python
{
    'ema_short': 10,
    'ema_mid': 30,
    'ema_long': 60,
    'holding_period': 3,
    'exit_on_macd_cross': True,   # 快速止盈
}
```
